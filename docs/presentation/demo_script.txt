[0:00 - 0:15] INTRODUCTION
----------------------------
"Hello, and welcome to our Music Genre Classification project demo.

[0:15 - 0:30] DATASET
-----------------------
"We trained our models on the GTZAN dataset, which contains 10,000
three-second audio segments, equally distributed across all 10 genres.
Each segment is represented by 57 audio features extracted using librosa,
including MFCCs, spectral centroid, and zero crossing rate."

[0:30 - 0:55] FEATURE ANALYSIS
--------------------------------
"We performed a comprehensive feature analysis using mutual information
scoring and ablation studies. Our top 3 features are:
  - MFCCs: which capture timbral texture
  - Spectral Centroid: which measures brightness
  - Zero Crossing Rate: which detects percussiveness

We also identified features to exclude, such as tempo and RMS energy,
which are unreliable for genre classification."

--------------------------------
"We implemented three models:

the power of deep learning for audio classification."
